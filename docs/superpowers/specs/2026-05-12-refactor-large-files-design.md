# Рефакторинг больших файлов

## Цель

Разделить три больших файла на меньшие компоненты для улучшения поддерживаемости кода.

## Файлы для рефакторинга

### 1. AdminBlog.tsx (1308 строк)

**Компоненты для создания:**
- `AdminBlogList.tsx` — таблица постов с поиском и фильтрацией
- `AdminBlogPostForm.tsx` — форма создания/редактирования поста  
- `AdminBlogPreview.tsx` — предпросмотр поста

**Изменяемые файлы:**
- `src/pages/AdminBlog.tsx` — импортирует новые компоненты

### 2. Blog.tsx (1275 строк)

**Компоненты для создания:**
- `BlogList.tsx` — список постов
- `BlogPost.tsx` — отображение поста
- `BlogSidebar.tsx` — сайдбар с категориями

**Изменяемые файлы:**
- `src/pages/Blog.tsx` — импортирует новые компоненты

### 3. CaseCard.tsx (1276 строк)

**Компоненты для создания:**
- `CaseCardBody.tsx` — основное содержимое карточки
- `CaseCardFooter.tsx` — действия (кнопки)
- `CaseCardEvents.tsx` — история событий дела

**Изменяемые файлы:**
- `src/components/CaseCard.tsx` — уже использует CaseCardHeader, добавить остальные

## Результат

Три файла уменьшатся до ~300-400 строк каждый.
Новые компоненты будут переиспользуемы.
