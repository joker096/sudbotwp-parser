# Changelog - Sud Bot Platform

## Версия 1.x - Обновление чата с юристом

### Дата: 2026-03-13

### Описание изменений
Чат с юристом переработан для удобства использования на мобильных и десктопных устройствах по примеру лучших мессенджеров (Telegram, WhatsApp).

---

## Основные изменения

### Мобильная версия (как в Telegram/WhatsApp)

1. **Компактная шапка**
   - Только аватар, имя юриста и статус "В сети"
   - Убрана лишняя информация о специализации юриста
   - Уменьшен размер аватара до 40px

2. **Навигация**
   - Кнопка "Назад" (стрелка) для возврата к списку чатов
   - Полноценное переключение между чатами

3. **Оптимизация пространства**
   - Кнопки звонков и видео скрыты на мобильных устройствах
   - Компактный баннер "Безопасная сделка" - сокращен до минимума
   - Кнопка прикрепления файлов в виде иконки, а не текста

4. **Список чатов**
   - Отображается как отдельный экран на мобильных
   - Показывает имя юриста, последнее сообщение, время и непрочитанные

---

### Десктопная версия

1. **Упрощенная шапка**
   - Компактное отображение без лишней информации
   - Иконки для действий: звонок, видео, меню

2. **Боковая панель**
   - Сохранена с полным списком чатов
   - Показывает онлайн-статус юристов
   - Непрочитанные сообщения выделены

3. **Баннер безопасной сделки**
   - Более информативный на десктопе
   - Кнопка раскрытия для подробностей

---

## Технические улучшения

1. **TypeScript**
   - Добавлен интерфейс `Chat` для типизации данных чата
   - Добавлены типы для вложенных файлов

2. **React State**
   - Состояние `mobileShowChat` для переключения между списком и активным чатом
   - Состояние `selectedChat` для выбора текущего чата

3. **Адаптивный дизайн**
   - Использование классов `md:` для breakpoint
   - Оптимизированные отступы и размеры элементов
   - Скрытие/показ элементов в зависимости от размера экрана

4. **Новые компоненты**
   - Демо-данные чатов в массиве `DEMO_CHATS`
   - Функции `handleSelectChat` и `handleBackToChats`
